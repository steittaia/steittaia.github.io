Hexo常用指令

  • npm install hexo -g #安装Hexo

  • npm update hexo -g #更新Hexo

  • hexo init blog #新建初始化博客，在blog文件夹下

  • hexo n blog_name = hexo new blog_name #新建名为blog_name的文章

  • hexo g = hexo generate #生成博客网页

  • hexo s = hexo server #在本地端口预览

  • hexo d = hexo deploy #部署网页

  • hexo clean #清除缓存
